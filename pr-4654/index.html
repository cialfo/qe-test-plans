CDocs — Documents TC-CDOC-038 The Application year mismatch banner's trigger condition changed from grade-based (Grade 12, or unknown grade, via regex/roman-numeral parsing of `student.grade`) to graduation-year-based (`student.graduation_year == cycleYear + 1`). TC-CDOC-038's precondition ('a student flagged with an application year mismatch') no longer maps to a concrete data setup — it must be revised to require a student with graduation_year = current cycle year + 1, not a Grade 12 student. The banner renders on both the Documents and Submission tabs (CdocsProfileBannerComponent), so both surfaces are affected even though only one TC currently exists.
